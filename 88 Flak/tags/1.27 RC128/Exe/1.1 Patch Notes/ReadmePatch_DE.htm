Readme für den Freelancer Server-Patch

Zusammenfassung der Änderungen

Server-Änderungen

Es wurden Maßnahmen zur Verhinderung von Serverabstürzen hinzugefügt, einschließlich Abstürzen, die durch Client-Mods und einfache Cheats verursacht werden.

Die Protokolldatei wurde bereinigt. Unwichtige Protokolleinträge wurden entfernt, und weitere Einträge für Spieleraktionen wurden hinzugefügt.

Es wurden Maßnahmen zur Erkennung von Cheats hinzugefügt, einschließlich der automatischen Verbindungstrennung und Aufzeichnung mutmaßlicher Cheater.

Die Spielerdaten werden nun auch bei der Landung gespeichert. Bisher wurden sie nur beim Start oder bei einer Unterbrechung des Spiels gespeichert.

Der Server versucht nicht mehr, eine Verbindung zum globalen Server herzustellen, wenn die Option, den Server über das Internet verfügbar zu machen, deaktiviert ist.

Client-Änderungen

Fehlerbeseitigung bei der Erzeugung einer Multiplayer-Konto-ID, falls keine ID erzeugt wurde.

Beseitigung von Abstürzen, wenn Client und Server auf demselben Computer ausgeführt werden.

Beseitigung der falschen Verzögerungsanzeige, wenn Client und Server auf demselben Computer ausgeführt werden.

Änderungen bei der Protokollierung

Für das Erstellen und Löschen von Konten und Charakteren sowie für die Cheat-Erkennung wurde die Protokollierung erweitert.

Die Einträge für die An- und Abmeldung beim System wurden entfernt.

Protokolleinträge zu Informationszwecken (im Gegensatz zu Warnmeldungen und Fehlerprotokollierung) beginnen mit „LOG:“, damit sie leichter erkennbar sind.

Informationen zur Cheat-Erkennung

Die Verbindung von Spielern, die Cheats verwenden, wird automatisch unterbrochen. Vom Server wird eine persönliche Nachricht mit dem Namen des Cheaters an alle Spieler gesendet. Es wird ein Protokolleintrag mit dem Kontonamen des Cheaters erzeugt. Die persönliche Nachricht an alle Spieler wird in der Datei CHEATMESSAGE.TXT im selben Verzeichnis wie das Programm FLSERVER.EXE gespeichert. Hierbei handelt es sich um eine Unicode-Textdatei, die vom Serveradministrator ggf. bearbeitet werden kann. Die maximale Länge der Cheat-Nachricht beträgt 80 Zeichen.

Cheater werden ebenfalls in der Datei CHEATERS.LOG im Verzeichnis mit den Multiplayer-Konten verzeichnet. Jeder Eintrag enthält das Datum und die Uhrzeit der Erkennung, den Namen des Charakters, den Konto- und den Spielernamen.

Cheater werden nicht automatisch gesperrt. Dies bleibt dem Serveradministrator überlassen.
